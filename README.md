# Skattefilerne

**Dine skattekroner. Deres spild. Grav i skandalerne, konsulentregningerne og de ansvarlige.**

## [Grav i Skattefilerne](https://asworbix.github.io/TransparentTax/)

---

### Hvad er det?

En investigativ webapp der afslører hvordan den offentlige sektor spilder dine skattekroner:

1. **IT-skandalerne** - Milliarder spildt på projekter der aldrig virkede. Navnene, tallene og hvem der trak i trådene.
2. **Konsulentregningen** - 4+ mia. kr. om året til konsulenter. Hvem får pengene, og hvad leverer de?
3. **Kontanthjælp** - 86.100 modtagere, 18,5 mia. kr. om året. Hvad virker, og hvad virker ikke?
4. **Følg pengene** - Budget vs. virkelighed, kvartalsforbrug og kendte problemer i den offentlige sektor.
5. **AI som løsning** - Hvordan moderne AI-teknologi kan forhindre fremtidens skandaler.

### Hvorfor?

Skandalerne gentager sig fordi systemet ikke har incitament til at ændre sig. Skattefilerne giver dig dataen, kilderne og overblikket til at kræve svar. Alt er offentligt tilgængeligt, verificerbart og åben kildekode.

### Navngivne aktører

Skattefilerne navngiver de konsulentfirmaer der har fået milliarder for fejlslagne projekter: Netcompany, KMD, CSC/DXC, Deloitte, Accenture, McKinsey m.fl. Og de politikere og embedsmænd der godkendte aftalerne.

### Data og kilder

- IT-skandaler: Rigsrevisionen, Version2, DigiTech/Ingeniøren
- Konsulentdata: Computerworld, Kammeradvokaten, offentlige udbud
- Skattesatser: skat.dk, Skatteministeriet (2025)
- Budget: Finansloven 2024/2025, Statsregnskabet 2024 (oes.dk)
- Kontanthjælp: Danmarks Statistik, DA, OECD, Beskæftigelsesministeriet
- Bevillingsregler: Budgetvejledning 2021 (oes.dk)

### Teknologi

Ren HTML/CSS/JS. Ingen frameworks, ingen build step, ingen afhængigheder. Åbn `index.html` og det virker.

### AI og Orbix Core

Skattefilerne peger på [Orbix Core](https://orbixcore.ai) som eksempel på hvordan AI-drevet softwareudvikling kan løse de systemiske problemer der forårsager offentlige IT-skandaler. Hurtigere levering, færre fejl, fuld gennemsigtighed.
